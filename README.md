#### Works with
* Almost any linux based shell.
* Cygwin shell on Windows.
* Git Bash on Windows.

#### Requirements
* cURL
* ffmpeg
* Some patience.

#### Setup Instructions
* If you are on linux, ensure that `hls_download.sh` is chmodded to `755`.
* Run `./hls_download.sh` with appropriate parameters.
